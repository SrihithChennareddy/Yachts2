# Naute Asset Naming Format

## Folders
- `assets/videos/`
- `assets/images/`

## Required naming
- Launch page hero background video: `assets/videos/video-01.mp4`
- Your Page (`yacht-life.html`) hero background video: `assets/videos/video-02.mp4`
- Article/list images: `assets/images/image-01.jpg` ... `assets/images/image-10.jpg` (or `.png`, `.webp`, `.svg`)
- Optional per-card hover preview videos (for index cards):
  - `assets/videos/image-01-preview.mp4`
  - ...
  - `assets/videos/image-10-preview.mp4`

The index page is already wired to these names.
