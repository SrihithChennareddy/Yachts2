# Naute Asset Naming Format

Media now loads from the repository root (not from `assets/`).

## Required naming
- Launch page hero background video: `video-01.mp4`
- Your Page (`yacht-life.html`) hero background video: `video-02.mp4`
- Article/list images: `image-01.jpg` ... `image-10.jpg`
- Optional per-card hover preview videos:
  - `image-01-preview.mp4`
  - ...
  - `image-10-preview.mp4`

If you upload files with these names directly in the repo root, pages will pick them up automatically.
